## 🙏 About

**PANDIT JI** is a comprehensive, single-file Hindu spiritual platform built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, no backend required. It covers everything a Hindu devotee needs in one beautiful, fast-loading app.

## 🚀 Deploy to Vercel

### Step 1 — Prepare GitHub Repository

```bash
# Create a new folder on your computer
mkdir panditji
cd panditji

# Copy HinduSpiritualPlatform_v3.html into this folder
# Rename it to index.html
mv HinduSpiritualPlatform_v3.html index.html

# Initialize git
git init
git add .
git commit -m "🕉️ Initial commit — PANDIT JI v4.0"
```

### Step 2 — Push to GitHub

```bash
# Go to github.com → New Repository
# Name it: panditji  (or any name you like)
# Keep it Public
# Do NOT add README (we already have one)

# Then run:
git remote add origin https://github.com/YOUR_USERNAME/panditji.git
git branch -M main
git push -u origin main
```

### Step 3 — Deploy on Vercel

1. Go to **[vercel.com](https://vercel.com)** → Sign in with GitHub
2. Click **"Add New Project"**
3. Select your **panditji** repository
4. Vercel will auto-detect it as a static site
5. Click **"Deploy"** — done in 30 seconds! ✅

### Step 4 — Custom Domain (Optional)

In Vercel Dashboard → Your Project → Settings → Domains
- Add your domain like `panditji.com`
- Vercel gives free SSL automatically

## ⚙️ AI Assistant Setup (Optional)

The AI Assistant uses the **Anthropic Claude API**. It works in two ways:

**Without API Key** — The app has intelligent built-in responses for common questions (OM, Karma, Dharma, Shiva, Krishna, Gita, Meditation, etc.)

**With API Key (full AI)** — For full Claude AI responses, the API call is made directly from the browser using:
```
anthropic-dangerous-direct-browser-access: true
```
This header enables direct browser access. The API key must be added to the fetch call in the source code.

> ⚠️ **Note:** For production, use a backend proxy to keep your API key secure. The direct browser method is suitable for personal/demo use.
